# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

## [1.0.0] - 2024-03-20

### Ajouté
- Site de présentation complet avec design moderne
- Documentation technique détaillée
- Configuration GitHub Actions pour CI/CD
- Templates pour les issues et pull requests
- Configuration Git LFS pour les fichiers volumineux
- Support GitHub Pages pour la documentation
- Système de badges pour le README
- Labels GitHub personnalisés

### Fonctionnalités principales
- Interface utilisateur responsive avec Tailwind CSS
- Intégration avec l'API SIV
- Utilisation de GPT-4 pour l'analyse
- Automatisation via n8n
- Base de données Supabase
- Tableau de bord complet
- Système d'estimation intelligent
- Assistant IA intégré

### Technique
- React avec TypeScript
- Tailwind CSS pour le style
- Shadcn UI pour les composants
- React Router pour la navigation
- Tanstack React Query pour la gestion des données
- Configuration complète pour le développement
- Workflows GitHub Actions
- Documentation technique détaillée
