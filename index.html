Comprendre le fonctionnement d'ArgusAI

ArgusAI est une application développée durant mon stage, conçue pour estimer intelligemment la valeur d'un véhicule à partir de sa plaque. Le processus est entièrement automatisé grâce à une série de technologies combinées :

  • Webhook pour recevoir la requête utilisateur
  • GPT-4o pour extraire automatiquement les informations du véhicule
  • API SIV pour récupérer les données officielles
  • Scraping Leboncoin via BrowseAI pour comparer à des annonces réelles
  • Filtrage & calcul pour proposer une estimation min / moyenne / max

Le tout est orchestré dans n8n, outil d'automatisation open-source.

Design moderne & technologies utilisées

ArgusAI dispose d'une interface claire, rapide et responsive. J'ai utilisé une stack web moderne pour offrir la meilleure expérience possible :

  • React + TypeScript pour le frontend
  • Tailwind CSS et Shadcn UI pour le design
  • React Router pour la navigation
  • Supabase pour la base de données et l'authentification
  • Tanstack React Query pour la gestion des données côté client

L'application est aussi optimisée pour le SEO (balises Open Graph, sitemap, metadata dynamiques).
